    线程状态¶

    操作系统线程的状态：

    • new
    • ready
    • running
    • waiting
    • terminated

    Java 中把 waiting 更加细化，分为三种：WAITING，TIMED_WAITING，BLOCKED，同时把 ready 和 running 合并为 RUNNABLE
